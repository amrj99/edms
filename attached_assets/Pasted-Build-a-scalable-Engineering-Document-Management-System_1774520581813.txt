Build a scalable Engineering Document Management System (EDMS) that can evolve into a multi-tenant SaaS platform similar in concept to Aconex, but simpler and optimized for medium-size companies.

Architecture Requirements
Frontend:
React + TypeScript

Backend:
Node.js using NestJS framework

Database:
PostgreSQL

Search Engine:
Elasticsearch

File Storage:
S3 compatible object storage

Authentication:
JWT based authentication

General Requirements
- Modular architecture
- Scalable for future SaaS
- Clean folder structure
- REST API
- Environment configuration
- Ready for Docker deployment
- Production-ready code

Core Modules (Phase 1)

1. Organizations
Organizations represent companies.
Fields:
- name
- type (Client, Consultant, Contractor, Subcontractor)
- contact details

Each organization can:
- own users
- participate in projects

2. Projects
Projects allow collaboration between organizations.

Project contains:
- dashboard
- documents
- correspondence
- tasks

3. User Management
Roles:
- Admin
- Project Manager
- Document Controller
- Reviewer
- Viewer

Implement Role Based Access Control (RBAC).

4. Document Management
Each document supports:
- document number
- title
- document type
- discipline
- revision
- status
- attachments
- metadata

Features:
- upload files
- version control
- revision history
- document preview
- folder structure

5. Metadata System
Metadata must be dynamic and configurable.

Examples:
- document type
- discipline
- prepared by
- issued by
- submission date

6. Correspondence Module
Create internal communication system similar to email.

Folders:
- Inbox
- Sent
- Draft
- Archive

Types:
- Transmittal
- Letter
- Memo
- RFI

Features:
- attachments
- response tracking
- audit trail

7. Workflow Engine
Configurable workflow:
Upload → Review → Approval → Issued

Workflow must support:
- comments
- approval
- rejection
- status tracking

8. Task Management
Tasks generated from:
- workflow
- correspondence

User dashboard:
- My tasks
- Pending approvals

9. Dashboard
User dashboard showing:
- pending approvals
- recent documents
- assigned tasks
- unread correspondence

10. Search
Use Elasticsearch for:
- full text search
- filter by metadata
- filter by project

Security
- RBAC
- document level permissions
- audit logs

Future SaaS Preparation
Design database to support:
- multi organization
- multi project
- tenant isolation

- enterprise clean layout
- sidebar navigation
- responsive design
- dark/light mode
