   Documents are automatically attached when creating a transmittal and cannot be removed.

2. Global Documents Search
   Search works inside a project but not in the global Documents view.
   Please:

* Search by number, title, filename, description

* Respect org permissions

3. Organization Name Input Bug

4. User ↔ Organization Assignment

* Members tab in Organization edit
* Organization selector in user edit
* Users must belong to an organization
