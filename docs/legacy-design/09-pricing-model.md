# Pricing Model — ArcScale EDMS
> Source: `التسعيير.odt`  
> Date: ~2026-04

## Product Positioning
ArcScale EDMS is not just Document Storage — it is an EDMS + Project Communication + Governance System. Value is high; pricing must be smart (not just cheap).

## Recommended Model: Hybrid (per user + plan + add-ons)

### Plans

| Plan | Price | Target | Includes |
|------|-------|--------|----------|
| Starter | 45 AED/user/month | Small company, internal only | Documents only, basic versioning |
| Basic | 70 AED/user/month | Small team, first project | Documents + Transmittals + Correspondence |
| Professional | 95 AED/user/month | Real projects | + Workflows + SLA + AI + Packages |
| Enterprise | 120 AED/user/month | Large companies | + API + SSO + Advanced analytics + Full governance |
| Custom | By agreement | Large orgs, government | On-premise, dedicated support, custom workflows, migration, training |

### Add-ons

| Service | Price |
|---------|-------|
| Extra storage | 20 AED / 50GB |
| Extra project | 30 AED |
| Data migration | By agreement |
| Training | 1,000–5,000 AED |
| Priority support | 500 AED/month |

## Value Ladder Philosophy
Sell the stage, not just features:
- Starter → Internal organization
- Basic → Communication management
- Professional → Project management
- Enterprise → Full governance
